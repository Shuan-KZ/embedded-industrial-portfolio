# GEC6818 四天嵌入式实训 — 智能语音交互系统

湖南科技大学 | 自动化四班十组 | 组长李宏欣 + 匡志琪 | 2026年4月

## 项目概述

在 GEC6818 ARM Cortex-A9 开发板上，基于 Linux + C 语言实现一个**语音+触控双模态智能交互系统**。涵盖嵌入式 Linux 开发全流程：交叉编译、LCD 驱动、触摸屏驱动、串口通信、多线程、网络编程、传感器集成。

| 组员 | 负责 | 视频角色 |
|------|------|---------|
| 李宏欣 (组长) | 硬件驱动：event.c, lcd.c, sensor.c | 操作开发板 |
| 匡志琪 | 系统集成+通信：common.c, voicectl.c | 录制全部5个验收视频 |

> 与项目1——工业设备监控系统（同仓库）呼应：均负责系统集成、通信协议、主控逻辑。

## 终期项目

### 系统架构
```
语音指令 → 录音(arecord) → PCM → TCP(54321) → Ubuntu语音识别引擎
                                                      ↓ XML结果
触摸手势 → /dev/input/event0 ──→  GEC6818开发板 ←── 指令ID
                                 │
                    ┌────────────┼────────────┐
                    ↓            ↓            ↓
                LCD显示      传感器采集    自动轮播
              (800×480)    (GY39/US100/    (3秒切换)
                            SR501)
```

### 核心功能
- **12条离线语音指令**：开窗/开门/开灯/关窗/关门/关灯/上一张/唱歌/你好/小雷(唤醒)/退出
- **4种触摸手势**：左滑下一张、右滑上一张、上滑传感器面板、下滑回图片
- **多传感器采集**：GY39光照/温湿度/气压/海拔 + US-100超声波测距 + SR501红外人体感应
- **3秒自动轮播**：无操作时自动切换图片，触摸或语音打断后重置计时
- **双线程并发**：触摸监听线程 + 语音识别线程，互斥锁保护共享状态

### 技术亮点
- ARM Linux framebuffer 直接显存操作，mmap 内存映射
- UART 串口协议解析（GY39 帧同步、US-100 超声波）
- Linux input 子系统触摸坐标采集+方向向量计算
- 离线语音识别 BNF 语法 + 置信度过滤（<30% 拒识）
- TCP Socket 通信 + XML 解析（libxml2）
- SR501 红外内核驱动加载（insmod infrared_drv.ko）

## 运行方式

1. Ubuntu 虚拟机启动语音识别引擎：`./asr_record_demo`
2. 开发板通过 SecureCRT 串口连接（115200bps）
3. 编译：`arm-linux-gcc *.c -o voicectl -Iinc -Llib -lasound -lxml2 -lpthread -lz`
4. rx 传输到开发板，`chmod +x voicectl`
5. 运行：`./voicectl <Ubuntu-IP>`
