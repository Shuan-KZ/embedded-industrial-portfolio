# 智能语音交互系统 设计说明

**组员**: 李宏欣(组长)、匡志琪  
**班级**: 自动化四班十组  
**平台**: GEC6818 开发板 + Ubuntu 虚拟机

---

## 1. 设计目标

实现一个 **语音 + 触控** 双模态智能交互系统：
- 语音控制（12 条指令，离线识别）
- 触摸手势控制（左/右/上/下滑动）
- LCD 显示（BMP 图片、背景色切换）
- 多传感器数据采集（GY39 光照/温湿度/气压/海拔、SR501 红外人体感应）
- 3 秒自动轮播模式
- 双线程并发架构

## 2. 系统结构

```
┌──────────────┐     TCP(54321)     ┌──────────────┐
│ Ubuntu 虚拟机  │ ←───────────────→ │ GEC6818 开发板 │
│ asr_record_   │   PCM音频/XML结果  │  voicectl     │
│ demo          │                    │               │
└──────────────┘                    │ LCD (/dev/fb0)│
                                    │ 触摸屏        │
                                    │ GY39 传感器   │
                                    │ SR501 红外    │
                                    └──────────────┘
```

## 3. 软件分层

| 文件 | 功能 |
|---|---|
| `voicectl.c` | 主程序: 双线程、指令分发、自动轮播、欢迎界面 |
| `event.c/h` | 触摸屏驱动: 四向滑动识别 (左/右/上/下) |
| `lcd.c/h` | LCD 驱动: framebuffer、BMP 显示、像素绘制 |
| `sensor.c/h` | 传感器: GY39(光照/温湿度/气压/海拔)、SR501 红外 |
| `common.c` | 网络通信: TCP socket、PCM 传输、XML 解析 |

## 4. 语音指令 (cmd.bnf)

| 指令 | ID | 执行动作 |
|------|----|---------|
| 开窗 | 1 | LCD 天蓝色背景 |
| 开门 | 2 | LCD 金色背景 |
| 开灯 | 3 | LCD 白色背景(模拟开灯) |
| 关窗 | 4 | LCD 浅钢蓝色背景 |
| 关门 | 5 | LCD 暗金色背景 |
| 关灯 | 6 | LCD 暗色背景(模拟关灯) |
| 上一张 | 7 | 切换到上一张 BMP 图片 |
| 唱首歌给我听 | 8 | 六色闪烁灯光秀 |
| 你好 | 20 | LCD 浅绿色背景 |
| 小雷/小雷小雷 | 100 | 唤醒词: 屏幕闪烁响应 |
| 退出/不玩了 | 999 | 优雅退出程序 |

## 5. 触摸手势

| 手势 | 功能 |
|------|------|
| 左滑 | 下一张图片 |
| 右滑 | 上一张图片 |
| 上滑 | 显示传感器数据面板 |
| 下滑 | 回到图片模式 |

## 6. 多线程设计

- **触摸线程**: 持续监听 `/dev/input/event0`，识别滑动方向
- **语音线程**: 录音 → 发送 PCM → 接收 XML → 解析指令 ID
- **主线程**: 自动轮播监测 (3 秒无操作自动切换)
- **互斥锁**: 保护图片索引、显示模式、自动轮播状态

## 7. 运行流程

1. Ubuntu 启动 `./asr_record_demo`
2. 开发板运行 `./voicectl 192.168.2.10`
3. LCD 显示欢迎界面
4. 进入自动轮播模式
5. 触摸或语音打断轮播，执行对应操作
6. 说出"退出"结束程序
