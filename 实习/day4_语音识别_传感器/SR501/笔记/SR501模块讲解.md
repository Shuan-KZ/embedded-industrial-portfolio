



### 模块

接线：
			VCC   ----    右上方从上往下数第三个串口的VCC(5V)
			OUT   ----    右上方从上往下数第三个串口的RX引脚
			GND   ----    右上方从上往下数第三个串口的GND



使用

1、将infrared_drv.ko下载到开发板的根目录下   (根目录就是  /  这个目录)

2、开发板中输入指令insmod 	infrared_drv.ko
		--> 注意： 重启开发板之后需要重新进行安装

3、用代码进行测试

````c

	int fd;
	int ret;
	int infrared_flag=0; 
	//打开设备文件---/dev/infrared_drv
	fd = open("/dev/infrared_drv", O_RDWR); 
	if(fd < 0)
	{
		perror("open /dev/infrared_drv");
		return -1;	
	}
	while(1)
	{	
		ret = read(fd,&infrared_flag,4);		
		if (ret < 0)
		{
			perror("read /dev/key_drv error\n");
			return -1;
		}

		printf("infrared flag = %d\n",infrared_flag);
		sleep(1);
			
	}
	close(fd);

infrared_flag: 红外标志
	值为1： 表示在检测范围内有物体
	值为0： 没有在检测范围内检测到物体
````
